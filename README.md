# Fera Search

Fera is an open-source metasearch project created by **Himanshu** and **Neev Jani**.  
It is inspired by and built using components from **SearXNG**, a powerful privacy-focused metasearch engine.

## About Fera
Fera aims to provide:
- Simple and fast web searching  
- Privacy-focused querying  
- Custom UI and features  

## About SearXNG
[SearXNG](https://github.com/searxng/searxng) is an open-source metasearch engine that aggregates results from multiple search providers while protecting user privacy.  
Fera uses or is based on parts of SearXNG according to its license (AGPL-3.0).
